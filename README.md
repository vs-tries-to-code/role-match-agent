# AI Career Discovery Agent

An AI-powered career tool that analyses your resume and suggests the most relevant job roles for you — complete with skill match scores, resume readiness scores, and personalised improvement tips.

Built with [Streamlit](https://streamlit.io/) and [Google Gemini 2.5 Flash](https://deepmind.google/technologies/gemini/), using [Google Antigravity](https://antigravity.google/)

---

## What it does

Upload a PDF resume and the agent will:

- Extract and parse the resume content into structured text
- Send it to Gemini 2.5 Flash with a role-matching prompt
- Return **3 job role suggestions** tailored to your skills and project experience
- Display for each role:
  - **Skill match %** — how well your skills align with the role
  - **Resume score %** — how ready your resume is for that role
  - **Justification** — why the role fits based on your background
  - **Improvement tip** — one actionable suggestion to strengthen your candidacy

This is especially useful for beginners and career-changers who may not be aware of all the roles available to them.

## Setup and installation

### 1. Clone the repository

```bash
git clone <repo-url>
cd <repo-folder>
```

### 2. Create a virtual environment

```bash
python -m venv venv
```

Activate it:

- **Mac/Linux:** `source venv/bin/activate`
- **Windows:** `venv\Scripts\activate`

### 3. Install dependencies

```bash
pip install -r requirements.txt
```


### 4. Set up your API key

Create a `.env` file in the root of the project:

```
API_KEY=your_google_gemini_api_key_here
```


### 5. Run the app

```bash
streamlit run agent.py
```

The app will open automatically in your browser at `http://localhost:8501`.

---

## Usage

1. Open the app in your browser
2. Click **Browse files** and upload your resume as a PDF
3. Wait a few seconds while the agent analyses your resume
4. View your 3 suggested roles with match scores and tips

---

## Known limitations

- Only PDF resumes are supported at this time
- Role suggestions are generated by an LLM and should be treated as guidance, not guarantees
- Scoring percentages are AI-estimated and not based on a validated scoring framework

---

## Scope for improvement

- **Multi-agent architecture** — separate agents for PDF parsing and role suggestion for better modularity and fault isolation
- **Improved scoring logic** — more robust, explainable percentage fit calculation using weighted skill matching against role taxonomies
- **Live job market integration** — connect to job board APIs (LinkedIn, Indeed, Reed) for demand-aware, real-time role recommendations
- **Personalised upskilling roadmap** — gap analysis between current skills and target role requirements, with structured course and certification suggestions
